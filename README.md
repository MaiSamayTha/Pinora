# Pinora

## Overview
Pinora is a minimal Next.js 14 + TypeScript starter focused on a clean foundation for future feature work.

## Tech
- Next.js 14 (App Router)
- React 18 with TypeScript
- ESLint + Prettier

## Getting Started
1. pnpm i
2. pnpm dev
